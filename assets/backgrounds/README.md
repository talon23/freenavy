# Page backgrounds

Each module has its own local background path in `config.js` and in `background-manifest.json`.

The supplied SVG files are original themed placeholders, not extracted Star Citizen artwork. To install final artwork:

1. Optimise the image as WebP, ideally 1920×1080 or larger.
2. Place it in this directory.
3. Update the relevant path in `config.js`.
4. Keep important visual subjects away from the left edge, where the navigation sidebar sits.

The CSS automatically adds dark veils and blurred interface panels to keep text readable.
